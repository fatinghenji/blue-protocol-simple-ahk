# 蓝色协议ahk
很懒，简化一些蓝色协议的操作。

目前只有法师。~因为我只玩了法师~

# 使用方法
1. 安装ahk
2. 下载，双击ahk文件运行
3. 激活和停用的时候会有tts，请确保安装了windows对应（英语）的语言包

| 功能  | 启用/禁用 | 触发 |
| ------------- | ------------- | ------------- |
| 一键Debuff  | 上箭头 | control + W |
| 一键MP恢复 | 下箭头 | control + S |
| 一键法师跳  | 左箭头 | control + A |

# 已知问题
